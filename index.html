What is HTML?

HTML or HyperText Markup Language is a powerful yet easy-to-understand language utilized globally for website creation. It consists of markup symbols or codes embedded in files for Internet display, instructing web browsers on rendering text and images.

What are the fundamentals of HTML?

At its core, HTML is a series of short codes typed into a text file. These are the tags that power HTML’s capabilities. The text is saved as an HTML file and viewed through a web browser. The browser reads the file and translates the text into a visible form, as directed by the codes that the author used to write what becomes the visible rendering.

Writing HTML requires tags to be used correctly to create the author’s vision. Tags separate normal text from HTML code. Tags, found between angle brackets, enable graphics, images, and tables to show on a webpage. Different tags perform different functions. The most basic tags apply formatting to text. To make web interfaces dynamic, Cascading Style Sheets (CSS) and JavaScript may be used. CSS makes webpages more accessible, and JavaScript adds power to basic HTML.

The basic tags of HTML

Every tag and code of the HTML language uses "<>" in the beginning of the tags. Without it, an error will occur.

Document Structure

  • !DOCTYPE - Defines the HTML version
  • html - The root element of the page
  • head - This contains the meta info
  • Title - This sets the title for your browser
  • body - This is where you input your main content for your browser

Text and Content

  • h1 to h6 - These are headings
  • p - This is for a paragraph
  • br - This is for a Line break
  • hr - This is for a Horizontal line
  • strong - This is to make text Bold
  • em - This is to make text Italic
  • a href="" - This is to make a link
  • img src="" alt="" - This is to input an image for your website

Lists

  • ul - This is to make an unordered list (bullets)
  • ol - This is to make an ordered list (numbers)
  • li - This is used inside the ul or ol (to list items)

Layout

  • div - This is to make a block-level container
  • span - This is to make an inline container

Table

  • table - This is to make a table
  • tr - This is for a table row
  • th - This is for a table header cell
  • td - This is for a table data cell

Forms

  • form - Is for making a form
  • input - Input field
  • textarea - for a multi-line text input
  • button - This is to make a button
  • label - This is to label an input
