Introduction to CSS

Cascading Style Sheets is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

What is "The Cascade"?

The cascade is the process CSS uses to decide which styles to apply when more than one rule targets the same element.

Concepts:

Specificity in CSS determines which rule wins when multiple rules target the same element. Each type of selector has a point value. The more specific the selector, the higher its score.

CSS Syntax and its parts

  • Selector - Specifies which HTML elements to style (p, .class, #id)
  • Property - The aspect of the element you want to change (color, font-size)
  • Value - The setting for the property (red, 16px, etc.)
  • Declaration - A combination of property and value (color: red;)
  • Declaration Block - One or more declarations wrapped in {}

Common Categories of CSS Properties

  • Text Properties - Control the appearance, alignment, spacing, transformation, and decoration of text content in elements.
  • Box Model Properties (Spacing & Size) - Define the space around elements and how their size is calculated, based on the CSS box model.
  • Background Properties - Control the background layer of elements, including color, images, position, and repetition behavior.
  • Layout & Display Properties - Control how elements are displayed and how they interact with other elements in the flow of the document.
  • Flexbox Properties - Used with display: flex, these properties allow for flexible, responsive layouts.
  • Animation & Transition Properties - Define motion and visual effects by animating changes in CSS properties.
  • Visibility & Effects - Manage the visibility of elements and apply visual effects, such as transparency, shadows, and filters.
  • Color - Specify a color (white, black, transparent, etc.)
  • Display/Layout - block, inline, inline-block, flex, grid, none
  • Positioning - static, relative, absolute, fixed, sticky
  • Font/Style - bold, italic, normal, uppercase, lowercase, capitalize
  • Units - px, em, rem, %, vh, vw, fr

Grid System

A CSS Grid System is a layout method in CSS used to create web page layouts based on a grid. It allows for precise control over rows and columns, making complex designs much easier to implement and manage.

  • Grid Container - Contains one or more grid items arranged in columns and rows.
  • Grid Items - A grid container that contains one or more grid items.
  • Grid Lines - The dividing lines in a grid used to control layout.
  • Grid Tracks - A row or column in the grid, formed by the space between two adjacent grid lines.
  • Grid Cells - The smallest area in the grid layout — one "box" in the grid.
  • Grid Area - The total space a grid item spans — across columns and rows.
  • Grid Template - The structure/blueprint of the grid layout.
  • Gap (Gutter) - The space between grid items.
  • Nested Grids - Grid within a Grid.
  • Media Queries (Responsive Design) - Grid systems often adapt to different screen sizes using media queries.
